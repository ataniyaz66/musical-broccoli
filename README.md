## Description
Music Hub is a full-stack web application that allows users to manage a music collection.
The project demonstrates backend API development, MongoDB integration, and a production-ready web interface.

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- HTML, CSS, JavaScript
- Fetch API

## Features
- Create new songs
- View all songs
- Update existing songs
- Delete songs
- Fully functional web UI
- Deployed to production
